Sobre a tag HEADER

A tag HEADER é utilizada para definir o cabeçalho de uma página. Ela pode conter elementos como logotipos, menus de navegação, formulários de busca, etc.

Sobre a tag NAV

A tag NAV é utilizada para definir a navegação de uma página. Ela pode conter elementos como links para outras páginas, links para outras partes da mesma página, etc.

Sobre a tag MAIN

A tag MAIN é utilizada para definir o conteúdo principal de uma página. Ela pode conter elementos como cabeçalhos, parágrafos, imagens, etc.

Sobre a tag SECTION

A tag SECTION é utilizada para definir seções de uma página. Ela pode conter elementos como artigos, parágrafos, imagens, etc.

Sobre a tag ARTICLE

A tag ARTICLE é utilizada para definir artigos de uma página. Ela pode conter elementos como cabeçalhos, parágrafos, imagens, etc.

Sobre a tag FOOTER

A tag FOOTER é utilizada para definir o rodape de uma página. Ela pode conter elementos como parágrafos, links, etc.
